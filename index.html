    • O que o git e o github?



    O github é uma plataforma utilizada por diversos programadores em todo mundo nele você pode publicar os seu códigos de programas e assim gerar um histórico
    das versões utilizadas, e assim tendo um controle sobre o que você já fez e podendo recuper o código em caso de perda do código



    Comandos do git

    • git status: verifica as propriedades do site
    • clear: limpa as informações na tela
    • git add "documento/pasta/arquivo": acessa o local do arquivo
    • git commit -m "aaaaaaaaa": adiciona um commit/"descrição" da versão
    • git push origin main: envia as informações ao site do github
